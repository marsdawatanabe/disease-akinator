# 病気アキネーター デザイン方針書

## 採用案: B案「フレンドリー・カード」

## デザインシステム: The Curated Sanctuary

### カラー
- Primary: #005c55 → #0F766E（グラデーション、135度）
- Background: #F7F9FB（Slate-50系）
- Surface: #FFFFFF（カード背景）
- Surface Container Low: #F2F4F6（セクション背景）
- Text: #191C1E（on-surface）、#3E4947（on-surface-variant）
- Error/Emergency: #BA1A1A
- Tertiary（暖色アクセント）: #7F4025 / #FFDBCE

### タイポグラフィ
- 見出し: Plus Jakarta Sans（Bold）
- 本文: Inter
- 日本語テキストのline-height: 1.8
- ベースフォントサイズ: 18px（体調不良者向け大きめ）

### レイアウト原則
- カード型UI全画面統一（角丸16px+）
- Bottom-Heavy UI（操作は画面下半分）
- 1問1画面
- ボーダー禁止 → 背景色のトーン差で区切る
- 余白を大きく取る（spacing-16以上）

### コンポーネント
- UIライブラリ: shadcn/ui + Lucide Icons
- ボトムシート: Vaul（モバイル）/ Dialog（PC）
- ボタン: pill型（round-full）、56px高さ
- カード: round-lg（2rem）以上
- プログレス: 円形リング + ステージテキスト

### アニメーション
- View Transitions APIでクロスフェード（控えめ）
- **prefers-reduced-motion 必須対応**
- 緊急画面はアニメーション排除（静的な赤背景）

### 画面構成

#### ① スタート画面
- 聴診器アイコン（ティール線画）
- アプリ名「病気アキネーター」
- 説明文カード
- 年齢セレクター + 性別セレクター（カード内）
- 「はじめる →」ボタン
- 免責テキスト

#### ② 緊急チェック画面
- 赤系背景（静的、アニメーションなし）
- チェックリスト一括表示
- 該当あり → 「119に電話」大きなボタン

#### ③ 質問画面
- 円形プログレスリング（40%等）+ 「候補を絞り込んでいます」
- 聴診器アイコン
- 質問テキスト（大きく太く）
- 「なぜこの質問？」展開セクション（電球アイコン付き）
- はい / いいえ / わからない ボタン（pill型、アイコン付き）
- 「← 前の質問に戻る」リンク

#### ④ 結果画面（Bento UI）
- ヒーローカード: 受診科（ティールグラデーション背景）
- 疾患候補カード × 3-5件（関連度バー付き）
- 展開可能な説明（「この病気について」「あなたの回答との関連」）
- 免責警告カード（アンバー背景）
- 「もう一度やり直す」ボタン

#### ⑤ フォールバック画面
- 「情報が不足しています」メッセージ
- 「まず内科を受診してください」
- 免責表示
